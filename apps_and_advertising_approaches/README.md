# apps_and_advertising_approaches

Индивидуальная часть Азимовой К. У.: адаптация методов из домена мобильных приложений и рекламы для детекции контрафактных карточек на маркетплейсе Ozon.

## Методы из обзора литературы

В качестве источников рассмотрены 4 работы:

- **Karunanayake et al.** (IEEE TMC, 2020). Feature Fusion на CLIP + Doc2Vec для детекции подменных приложений в Google Play.
- **Deng et al.** (ACM SaTS '24). Typosquatting-сигналы для контрафактных мини-приложений WeChat (MiniWuKong).
- **Mohsen et al.** (Systems and Soft Computing, 2022). Поведенческие метаданные продавца как ранний сигнал нарушений.
- **Nguyen et al.** (FADAML, Applied Intelligence 2025). Feature-engineering ценовых аномалий и текстовых статистических сигналов.

## Что в ноутбуках

`multimodal_counterfeit.ipynb` - адаптация метода Karunanayake et al. к Ozon. CLIP ViT-B/32 эмбеддинги изображений + TF-IDF SVD-50 текстов + CatBoost-агрегатор. Базовая мультимодальная архитектура Feature Fusion (Exp-1 в диссертации).

`improved_counterfeit_v3.ipynb` - улучшения поверх baseline: typosquat-признаки расстояния Левенштейна между названием товара и брендами категории (по аналогии с Deng), ценовые аномалии price_too_high (по аналогии с Nguyen), доменные текстовые сигналы, агрессивный class_weight, OOF-калибровка изотонией. Итоговая модель Exp-Combined.

## Результаты

Финальная модель **Exp-Combined** (614 признаков, индивидуальный seller-based сплит n_test = 19 818):

| Метрика | Значение |
|---------|----------|
| ROC-AUC | 0.9339 |
| PR-AUC | 0.7447 |
| Recall @ Precision >= 0.9 | 0.224 |

Получено 15-кратное улучшение R@P относительно лучшего одномодального baseline B2 (CatBoost на 38 табличных признаках продавца).

На общекомандном тестовом сплите (n_test = 58 410) модель Mobile-Adapted (упрощённая версия Exp-Combined без seller-специфичных признаков) даёт PR-AUC = 0.6453 и вкладывается в финальную HCDM композицию команды как канал p_mobile с весом 0.075.

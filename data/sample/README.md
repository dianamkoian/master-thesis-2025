# Sample-датасет

Маленький репрезентативный сэмпл основного датасета Ozon eCup 2025 для **быстрого запуска демо без скачивания** полных 900 MB.

## Файлы

| Файл | Размер | Строк | Назначение |
|------|--------|-------|------------|
| `train_sample.csv` | 478 KB | 500 | Случайный сэмпл `ozon_train.csv` (45 признаков + target) |
| `image_embeddings_sample.csv` | 202 KB | 500 | Случайный сэмпл `image_embeddings.csv` |

## Метод сэмплирования

```python
df.sample(n=500, random_state=42)
```

- Воспроизводимый seed: `42`
- Сэмплирование БЕЗ стратификации (наследует базовое распределение классов)
- Баланс target в `train_sample.csv`: 462 легитимных / 38 контрафакт (~7,6 % — близко к полному датасету)

## Когда использовать

✅ **Используй sample, если хочешь:**
- Быстро посмотреть структуру данных (`pd.read_csv` за секунду)
- Запустить демо-ноутбук без скачивания
- Проверить совместимость кода
- Сделать unit-тесты препроцессинга

❌ **Не используй sample для:**
- Обучения финальных моделей (нужен полный датасет с n=328k)
- Воспроизведения метрик из диссертации
- Bootstrap-CI и статистических тестов (нужен n > 50 000)

Для всех серьёзных задач — скачай полные данные через `python data/download.py` или вручную с Яндекс.Диска (см. [`../README.md`](../README.md)).

## Пример использования

```python

df = pd.read_csv("data/sample/train_sample.csv")
print(f"Shape: {df.shape}")
print(f"Columns: {list(df.columns)[:10]}")
print(f"Target balance: {df['resolution'].value_counts(normalize=True).to_dict()}")
```

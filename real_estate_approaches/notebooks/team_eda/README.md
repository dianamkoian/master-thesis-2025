# team_eda/ — командные EDA-ноуты

Этот каталог содержит развёрнутый разведочный анализ данных Ozon eCup 2025,
выполненный соавторами команды. Скопирован в финальную сборку
`Диана_ВКР_финал/` для полноты сопроводительных материалов диплома.

## Содержание и авторство

| Файл | Автор(ы) | Назначение |
|---|---|---|
| `01_eda_main_team.ipynb` (75 ячеек) | **Соня Красовская** (Part 1: базовый EDA), **Альбина Бахтиарова** (Part 2: углублённый EDA с MI / Mann-Whitney / производными признаками / seller-level / взаимодействиями) | Полный базовый и углублённый EDA датасета |
| `02_eda_text_extended_team.ipynb` (исходник) | **Соня Красовская / Альбина Бахтиарова** | Расширенный анализ текстовой модальности (Block A — длины; B — заполненность; C — brand-семантика; D — suspicious tokens; E — TF-IDF + LogReg top-N; F — brand↔name fuzzy similarity) |
| `02_eda_text_extended_executed_team.ipynb` (с выводами) | то же | Исполненная версия с outputs |
| `03_eda_images_team.ipynb` (19 ячеек) | **Соня Красовская / Альбина Бахтиарова** | EDA визуальной модальности (`photos_published_count`, CLIP-эмбеддинги и разделимость классов) |
| `text_eda_results.csv` | то же | Сводная таблица результатов текстового EDA |

## Связь с дипломом

Содержание этих ноутов сводно отражено в **Главе 2** диплома (раздел EDA).
Индивидуальная защитная версия с одиннадцатью визуализациями для слайдов
автора (Д. Мкоян) лежит отдельно в `notebooks/01_eda_defense.ipynb`.

Исходное место хранения этих ноутов — `new_diploma/notebooks/` в корне
репозитория; копия в финальной сборке нужна для удобства комиссии при
просмотре материалов в одной папке.
